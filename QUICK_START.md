# ⚡ Quick Start - NTA 8800 App op GitHub Pages

## 🎯 In 5 minuten online!

### Stap 1: GitHub Account
→ Ga naar **github.com/signup**
→ Maak gratis account

### Stap 2: Nieuwe Repository
→ Klik op groene **"New"** knop
→ Naam: `nta8800-app`
→ Public aanvinken
→ Klik **"Create repository"**

### Stap 3: Upload Bestanden
→ Klik **"uploading an existing file"** (blauwe link)
→ Sleep ALLE bestanden uit de `web-app` folder
→ Klik **"Commit changes"**

### Stap 4: Activeer Pages
→ Klik **"Settings"** (tandwiel)
→ Klik **"Pages"** (links in menu)
→ Source: **"main"** branch
→ Folder: **"/ (root)"**
→ Klik **"Save"**
→ Wacht 1 minuut

### Stap 5: Klaar! 🎉
Je URL is: `https://JOUW-USERNAME.github.io/nta8800-app/`

---

## 📱 Installeren op iPad

1. Open de URL in **Safari**
2. Deel-knop (□↑) → **"Zet op thuisscherm"**
3. Klaar! Werkt nu offline als app

---

## 📦 Wat te uploaden?

Upload deze bestanden uit de `web-app` folder:
- ✅ index.html
- ✅ manifest.json
- ✅ service-worker.js
- ✅ icon.svg
- ✅ icon-192.png
- ✅ icon-512.png
- ✅ .gitignore (optioneel)
- ✅ README.md (optioneel)
- ✅ netlify.toml (optioneel)

---

## 🆘 Problemen?

**"404 Not Found"**
→ Wacht 2-3 minuten, GitHub heeft even nodig

**"Settings → Pages niet zichtbaar"**
→ Check of repository "Public" is (niet Private)

**Meer hulp nodig?**
→ Lees: `GITHUB_PAGES_INSTRUCTIES.md` (volledig stappenplan)
